Hello Arduino World

12 December 2014

Before today, I didn't really know what an Arduino was. It sounded like an Italian shoe designer. I'd even had a friend mention Arduinos to me, and since I'm in school to be a developer, I pretended like I knew what he was talking about.

Well now I'm in the know, and they are so exciting! I did the requisite blinking-light program, but quick results have a way of making you feel passionate about a new experience early on.

Imagine the blinking. Hello World: 

def blink(led)  
  led.send(:on)
  sleep 0.1
  led.send(:off)
  sleep 0.1
end

Watch out, world-blinking domination is just around the corner!

What is the most amazing part of today is that this is one of those cases where the right path opens itself up in front of you. For a couple of months now, I've been interested in indoor gardening systems, and especially how technology can improve them. Arduino and its cousin, Raspberry Pi, are the solutions. I'm going to ask for a Raspberry Pi for Christmas. I'm going to start automating everything in my house. I'm going to help push these technologies further.

Here's a project that looks like a good start. I have a lemon tree I can try it out on. http://spin.atomicobject.com/2014/06/28/raspberry-pi-gardening/
For indoor gardening, I'm probably not going to hook up a hose. I might have a tank set up, and I'd probably want to send an email when the water level is getting too low. My mind is spinning with the possibilities of building out this project. I'm thinking of streaming data maps, and eventually adding remote user controls.

Everyone has to start somewhere, and this is it for me.

Laura Whalin

Web developer, mother, clean food advocate, writer, and circus-ring leader.
